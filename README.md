# AI 패킷 모니터
웹 기반 네트워크 트래픽 분석 및 시각화 시스템

## 개요
패킷을 수집하고, 저장, 분석하는 시스템입니다.

FastAPI 웹 프레임워크를 이용하여 만들었습니다.

## 상세
파이썬 코드가 패킷을 수집 해 오면, FastAPI를 거쳐 HTML/CSS로 요약하여 보여줍니다.

패킷 저장 버튼을 눌러 임의의 시간 동안 들어오는 모든 패킷을 sqlite db에 저장 할 수도 있고, 특별한 조건을 설정하여 조건에 맞는 패킷을 저장 할 수도 있습니다.

파이썬 코드와 HTML/CSS가 분리되어 작동하기 때문에, 포트포워딩 등 외부에서 접근할 수 있는 수단이 있다면 외부 환경에서도 접속 할 수 있습니다.

## 주요 기능
- 트래픽(초당 패킷 수) 확인
- 실시간 패킷 스니핑
- 패킷 저장 및 조회
- 패킷 강조 필터
- 의심 트래픽 자동 감지 및 알림
- 패킷 요약
  - 패킷 방향
  - 출발지
  - 목적지
  - 프로토콜
  - 패킷 크기
- AI 분석
  - Google Gemini API
- 외부 접속

## 개발 환경
- 언어 : Python, JavaScript
- 프레임워크 : FastAPI
- 패킷 처리 : Scapy, Npcap
- 프론트 : HTML, CSS

## 잡담
대학교 Lab-Scale 과목에서 진행하는 프로젝트입니다. 자유주제였지만 평소 네트워크에 관심이 꽤 있었기도 하고 생산성 있는 프로그램을 만들어보고 싶었기에 도전해봤습니다.

물론 기획만 제가 하고 대부분의 작업은 AI의 도움을 받아 하게 되었습니다. 작업을 하다 보니, 비전공자도 이렇게 쉽게 프로그램을 만들 수 있지 않을까 하는 생각이 들었습니다. AI의 발전이 지금 당장 과제를 하는데는 큰 도움이 되어주었지만 이후 저 같은 코딩쟁이의 수요가 확 줄어들어 구직 활동에 어려움이 될까봐 두렵기도 합니다.
